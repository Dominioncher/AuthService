# AuthService
Сервис управления пользователями для тестового задания IT Talent Program

### Структура проекта:

1. DB - Классы для работы с PostgreSQL
2. Services - Классы сервисов с бизнес-логикой
3. Controllers - Классы http контроллеров
4. Middleware - Классы промежуточных слоев Api
5. Migrations - Миграции EFCore 

### Локальный запуск проекта:
1. Поднять требуемое окружение через docker-compose up -d
2. Запустить серивис через Visual Studio
